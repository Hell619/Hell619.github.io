前言

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

准备安装Hexo的环境

在安装Hexo前，先安装

  • Node.js
    下载地址：

    https://nodejs.org/en/
  • Git
    下载地址：

    https://git-scm.com/
  • VScode
    下载地址：

    https://code.visualstudio.com/Download

    加速下载方式：

    先在vscode官网找到你要的版本，下载，然后将红框地址替换为：vscode.cdn.azure.cn
